## Run locally

```bash
git clone git@github.com:Soumik205/quotes.git
cd backend
node Server.js
cd ../frontend
pnpm install
pnpm dev
```

This will start the development server on http://localhost:5173

_You need to have Node installed and pnpm enabled._
